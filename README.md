# 農田水利法規測驗系統

115年 1月版 考古題模擬測驗，共 87 題選擇題。

## 啟動方式

```bash
cd 證券法規測驗
python server.py
```

啟動後會顯示本機和區網 IP，其他電腦在瀏覽器輸入該網址即可使用。

## 檔案結構

```
├── server.py          # 伺服器（port 8080）
├── index.html         # 首頁（設定、歷史紀錄）
├── quiz.html          # 測驗頁面
├── review.html        # 測驗結果 / 歷史紀錄詳情
├── browse.html        # 題庫瀏覽
├── wrong.html         # 錯題檢視（今日/過去）
├── questions.json     # 題庫
├── css/style.css      # 樣式
├── js/shared.js       # 共用 JS（複製、提示詞、元件）
└── data/
    └── history.json   # 作答歷史（自動產生）
```

## 功能

- 隨機出題 + 選項順序打亂
- 題號跳轉（可自由切換題目）
- **交卷後才對答案**（做題時不會顯示對錯）
- 完整報告：錯題 + 答對題目全部列出
- **一鍵複製（含 AI 提示詞）**，可直接貼給 ChatGPT / Claude
- 歷史紀錄：保留所有紀錄，可查看、單筆刪除
- 今日錯題 / 過去錯題分開統計
- 題庫瀏覽：可查看全部題目與答案
- 重做錯題模式
- 大字體設計，適合年長者使用
